# SRS-003
User Personas & Roles

## 1. مقدمة

يوضح هذا المستند جميع أنواع المستخدمين في منصة كلامي، وأهدافهم، وصلاحياتهم، وحالات الاستخدام الأساسية، ليكون مرجعاً لفريق التصميم والبرمجة.

## 2. الشخصيات الرئيسية (Personas)

| الدور | الهدف | المهام الرئيسية |
| --- | --- | --- |
| المستفيد | تنفيذ الخطة العلاجية | الاطلاع على الخطة، أداء التمارين، حضور الجلسات، متابعة التقدم |
| ولي الأمر | متابعة الطفل | إدارة بيانات الطفل، متابعة الالتزام، استلام الإشعارات |
| أخصائي التخاطب | إدارة العلاج | التقييم، إنشاء الخطة، توثيق الجلسات، إصدار التقارير |
| المشرف السريري | ضمان الجودة | مراجعة الحالات، التدقيق السريري، متابعة مؤشرات الجودة |
| مدير النظام | إدارة المنصة | إدارة المستخدمين، الصلاحيات، الإعدادات، التقارير التشغيلية |

## 3. مصفوفة الصلاحيات

## 4. حالات الاستخدام الرئيسية

- المستفيد يسجل حساباً ويبدأ الخطة العلاجية.

- ولي الأمر يتابع تقدم الطفل ويتلقى الإشعارات.

- الأخصائي يجري التقييم ويعتمد الخطة العلاجية.

- المشرف يراجع جودة التطبيق السريري.

- مدير النظام يدير المستخدمين والإعدادات.

## 5. متطلبات أمنية

- تطبيق مبدأ أقل صلاحية (Least Privilege).

- تسجيل جميع العمليات الحساسة في سجل التدقيق.

- التحقق من الصلاحيات قبل كل عملية.
