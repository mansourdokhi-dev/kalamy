# KALAMY-002
Screen Specification

## الجزء الرابع: لوحات التحكم والإدارة

### SCR-012 - لوحة الأخصائي

الغرض: إدارة الحالات والجلسات اليومية.

| الحقل/العنصر | النوع | إلزامي | ملاحظات |
| --- | --- | --- | --- |
| الحالات | Grid | نعم | قائمة المرضى |
| المواعيد | Calendar | نعم | اليوم والأسبوع |
| التنبيهات | Panel | لا | حالات تحتاج متابعة |

#### الأزرار

- فتح الحالة

- بدء جلسة

- إضافة ملاحظة

#### قواعد العمل

- يعرض فقط الحالات المخصصة للأخصائي.

API: GET /api/v1/clinician/dashboard

Navigation: إلى ملف المستفيد أو الجلسة
