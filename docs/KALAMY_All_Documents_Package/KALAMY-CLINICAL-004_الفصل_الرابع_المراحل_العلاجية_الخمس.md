# KALAMY-CLINICAL-004

## الفصل الرابع: المراحل العلاجية الخمس

## 1. الهدف

يوضح هذا الفصل المراحل العلاجية الأساسية في برنامج كلامي، وأهداف كل مرحلة، ومعايير الانتقال بينها، بما يضمن رحلة علاجية موحدة وقابلة للقياس.

### المرحلة الأولى: التهيئة والتوعية

بناء العلاقة العلاجية، تثقيف المستفيد والأسرة، تحديد خط الأساس، ورفع الدافعية.

#### الأهداف

- تحقيق أهداف المرحلة المحددة.

- قياس الأداء باستخدام مؤشرات كمية ونوعية.

- توثيق النتائج داخل المنصة.

#### معايير الانتقال

- تحقيق الحد الأدنى من مؤشرات النجاح.

- التزام المستفيد بالواجبات العلاجية.

- اعتماد الأخصائي للانتقال.

### المرحلة الثانية: اكتساب المهارات

تعليم تقنيات الطلاقة الأساسية مثل التحكم بالتنفس، البداية السهلة، والإطالة المناسبة.

#### الأهداف

- تحقيق أهداف المرحلة المحددة.

- قياس الأداء باستخدام مؤشرات كمية ونوعية.

- توثيق النتائج داخل المنصة.

#### معايير الانتقال

- التزام المستفيد بالواجبات العلاجية.

- اعتماد الأخصائي للانتقال.

### المرحلة الثالثة: التطبيق الموجه

تطبيق المهارات في مواقف علاجية متدرجة مع تغذية راجعة مستمرة.

#### الأهداف

- تحقيق أهداف المرحلة المحددة.

- قياس الأداء باستخدام مؤشرات كمية ونوعية.

### المرحلة الرابعة: التعميم

نقل المهارات إلى مواقف الحياة اليومية في المنزل، المدرسة، والعمل.

#### الأهداف

### المرحلة الخامسة: المحافظة ومنع الانتكاس

تعزيز الاستقلالية، متابعة دورية، وخطة للتعامل مع أي تراجع مستقبلي.

## 2. قواعد عامة

- قد يختلف زمن كل مرحلة حسب حالة المستفيد.

- يمكن العودة إلى مرحلة سابقة إذا استدعت الحالة.

- جميع القرارات العلاجية توثق داخل النظام.

- تتم مراجعة الخطة العلاجية بشكل دوري.

## 3. التكامل مع منصة كلامي

تعرض المنصة المرحلة الحالية للمستفيد، ونسبة الإنجاز، والأهداف المتبقية، وتقترح التمارين المناسبة لكل مرحلة مع تسجيل جميع مؤشرات الأداء لدعم القرار السريري.
