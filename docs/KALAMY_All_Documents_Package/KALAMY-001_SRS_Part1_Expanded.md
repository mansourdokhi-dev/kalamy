# KALAMY-001
Software Requirements Specification (SRS)
Part 1

هذه الوثيقة هي الجزء الأول من المواصفات التنفيذية لبرنامج كلامي، وهي موجهة لفريق التصميم والبرمجة.

## 1. معلومات الوثيقة

- اسم المنتج: كلامي (KALAMY)

- نوع المنتج: منصة علاج رقمي لاضطرابات الطلاقة والتلعثم

- الإصدار المستهدف: MVP 1.0

- الجمهور: فريق التصميم، فريق البرمجة، مدير المشروع، فريق الاختبارات

## 2. أهداف المنتج

- إدارة رحلة المستفيد من التسجيل حتى إنهاء العلاج.

- تمكين الأخصائي من إدارة الخطط العلاجية والجلسات.

- تقديم تمارين علاجية رقمية مع قياس الأداء.

- إصدار تقارير علاجية ومؤشرات تقدم.

## 3. أصحاب المصلحة

## 5. User Story US-001: إنشاء حساب

كمستفيد، أريد إنشاء حساب جديد حتى أتمكن من استخدام منصة كلامي.

### معايير القبول

- إدخال الاسم ورقم الجوال وكلمة المرور.

- التحقق من عدم تكرار رقم الجوال.

- إرسال رمز تحقق OTP.

- عدم تفعيل الحساب قبل التحقق.

### قواعد العمل

- إذا كان عمر المستفيد أقل من 18 سنة فيجب ربط ولي أمر.

- تسجل عملية إنشاء الحساب في سجل التدقيق.

## 6. User Story US-002: تسجيل الدخول

كمستخدم مسجل، أريد تسجيل الدخول للوصول إلى بياناتي.

### معايير القبول

- التحقق من صحة بيانات الدخول.

- إيقاف المحاولات مؤقتاً بعد خمس محاولات فاشلة.

- إصدار رمز جلسة آمن بعد نجاح الدخول.
