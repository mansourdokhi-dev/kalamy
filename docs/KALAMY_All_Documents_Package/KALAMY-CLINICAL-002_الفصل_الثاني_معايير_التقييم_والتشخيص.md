# KALAMY-CLINICAL-002

## الفصل الثاني: معايير التقييم والتشخيص

## 1. أهداف التقييم

يهدف التقييم إلى تحديد طبيعة اضطراب الطلاقة، وقياس شدته، وفهم العوامل المؤثرة عليه، وبناء خطة علاجية فردية قابلة للقياس.

## 2. محاور التقييم

- البيانات الديموغرافية والتاريخ المرضي.

- بداية التلعثم ومساره الزمني.

- التاريخ العائلي.

- البيئة المدرسية أو المهنية.

- المواقف الأكثر صعوبة في التواصل.

- السلوكيات الثانوية والتجنب.

- الأثر النفسي والاجتماعي.

- العوامل الطبية والسمعية واللغوية ذات العلاقة.

## 3. أدوات التقييم المقترحة

| الأداة | الغرض |
| --- | --- |
| المقابلة السريرية | جمع التاريخ المرضي وتحديد الاحتياجات. |
| استبيان المستفيد/ولي الأمر | قياس أثر التلعثم على الحياة اليومية. |
| عينة كلام | تحليل الطلاقة في مواقف مختلفة. |
| مقاييس الشدة | قياس خط الأساس ومتابعة التحسن. |
| الملاحظة السريرية | تحديد السلوكيات الثانوية والعوامل البيئية. |

## 4. مخرجات التقييم

- تحديد مستوى الشدة.

- تحديد الأهداف العلاجية الأولية.

- تحديد المرحلة العلاجية المناسبة.

- تحديد مؤشرات القياس الأساسية (Baseline).

- توصيات للأسرة أو المستفيد.

## 5. معايير الجودة

- إكمال جميع عناصر التقييم الإلزامية.

- توثيق النتائج داخل النظام.

- اعتماد التقييم من الأخصائي.

- إمكانية مقارنة النتائج في الزيارات اللاحقة.

## 6. القرار السريري

لا يبدأ البرنامج العلاجي إلا بعد اكتمال التقييم واعتماد الخطة العلاجية الأولية. ويُعاد التقييم عند حدوث تغير جوهري في الحالة أو عند نقاط المراجعة المحددة.
