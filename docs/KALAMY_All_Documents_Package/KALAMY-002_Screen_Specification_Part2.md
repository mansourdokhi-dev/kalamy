# KALAMY-002
Screen Specification

## الجزء الثاني: الملف الشخصي والتقييم والخطة العلاجية

### SCR-004 - الملف الشخصي للمستفيد

الغرض: إنشاء وتحديث الملف السريري.

| الحقل | النوع | إلزامي | ملاحظات |
| --- | --- | --- | --- |
| الاسم | Text | نعم |  |
| تاريخ الميلاد | Date | نعم |  |
| الجنس | Dropdown | نعم |  |
| بيانات ولي الأمر | Section | حسب العمر |  |

#### الأزرار

- حفظ

- التالي

#### قواعد العمل

- لا يمكن بدء التقييم قبل اكتمال الحقول الإلزامية.

API: GET/PUT /api/v1/patients/{id}

Navigation: التالي → التقييم الأولي

### SCR-005 - التقييم الأولي

الغرض: جمع البيانات السريرية وخط الأساس.

| الحقل | النوع | إلزامي | ملاحظات |
| --- | --- | --- | --- |
| الاستبيان | Form | نعم |  |
| عينة صوت | Audio | نعم |  |
| ملاحظات | TextArea | لا |  |

#### الأزرار

- حفظ

- اعتماد التقييم

#### قواعد العمل

- اعتماد التقييم يفتح إنشاء الخطة العلاجية.

API: POST /api/v1/assessments

Navigation: التالي → الخطة العلاجية

### SCR-006 - الخطة العلاجية
