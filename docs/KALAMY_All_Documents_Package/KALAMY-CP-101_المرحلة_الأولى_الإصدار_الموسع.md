# KALAMY-CP-101

## المرحلة الأولى: التهيئة والتوعية
الإصدار التنفيذي الموسع

هذه الوثيقة تمثل النسخة الموسعة للمرحلة الأولى من البروتوكول العلاجي لبرنامج كلامي. وهي مصممة لتكون أساساً لبناء النظام الرقمي وربط الجوانب السريرية بالشاشات وقواعد العمل.

## 1. الرؤية

تهيئة المستفيد وأسرته وأخصائي التخاطب لبدء رحلة علاجية قائمة على الأدلة والقياس.

## 2. أهداف المرحلة

- • بناء التحالف العلاجي.

- • رفع الوعي.

- • تحديد خط الأساس.

- • رفع الدافعية.

- • إعداد الأسرة.

- • اعتماد خطة البداية.

## 3. معايير الدخول

وجود تقييم أولي مكتمل، موافقة المستفيد أو ولي الأمر، عدم وجود موانع تحول دون بدء البرنامج.

## 4. معايير الاستبعاد المؤقت

الحالات الطبية أو النفسية غير المستقرة، الحاجة إلى تقييمات إضافية، أو عدم اكتمال البيانات.

## 5. رحلة المستفيد

- إنشاء الحساب → استكمال الملف → مشاهدة المحتوى التثقيفي → الإجابة عن الاستبيانات → تسجيل عينة كلام → مراجعة الأخصائي → اعتماد الجاهزية.

## 6. رحلة ولي الأمر

استكمال المحتوى التثقيفي، قراءة الإرشادات، تنفيذ الواجبات المنزلية، متابعة الالتزام.

## 7. رحلة الأخصائي

مراجعة البيانات، اعتماد التقييم، تحديد درجة الجاهزية، إنشاء خطة المرحلة الأولى، تحديد موعد المراجعة.

## 8. المحتوى التثقيفي

يشمل تعريف التلعثم، المفاهيم الخاطئة، دور الأسرة، أهمية الممارسة، آلية البرنامج، والتوقعات الواقعية.

## 9. الجلسات المقترحة

- الجلسة 1: التعارف وبناء الثقة.

- الجلسة 2: فهم التلعثم.

- الجلسة 3: تحديد الأهداف.

- الجلسة 4: مراجعة الجاهزية.

## 10. الواجبات المنزلية

مشاهدة المواد التعليمية، تسجيل مواقف الكلام، الإجابة عن أسئلة الانعكاس الذاتي، الالتزام بالتذكيرات.

## 11. مؤشرات الأداء

الالتزام، إكمال المحتوى، درجة الجاهزية، حضور الجلسات، رضا المستفيد.

## 12. قواعد العمل

- عدم الانتقال للمرحلة التالية قبل تحقيق معايير الجاهزية واعتماد الأخصائي.

## 13. محرك القرار السريري

يحسب درجة الجاهزية ويقترح الاستمرار أو إعادة بعض الأنشطة أو اعتماد الانتقال مع بقاء القرار النهائي للأخصائي.

## 14. متطلبات النظام

شاشة الجاهزية، لوحة متابعة، إشعارات، تقارير، سجل نشاط، صلاحيات.

## 15. حالات الاختبار

التحقق من منع الانتقال عند عدم اكتمال المتطلبات، إصدار التنبيهات، حفظ نتائج الجاهزية.

## 16. معايير الجودة

توثيق كامل، مراجعة دورية، تتبع جميع القرارات السريرية، حفظ السجل التاريخي.

## ملاحظة تنفيذية

هذه الوثيقة الموسعة ليست النسخة النهائية الكاملة ذات 100–150 صفحة. إعداد تلك النسخة يتطلب إنتاج كتاب كامل يتجاوز حدود رسالة واحدة، مع جداول تفصيلية ورسومات ومراجع ومئات قواعد العمل.
