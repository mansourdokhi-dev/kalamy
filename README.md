# Kalamy (كلامي)

[![Tests](https://github.com/mansourdokhi-dev/kalamy/actions/workflows/tests.yml/badge.svg)](https://github.com/mansourdokhi-dev/kalamy/actions/workflows/tests.yml)

Digital therapeutics platform for fluency disorders / stuttering, for the Arabic-speaking market.

- `backend/` — NestJS + Prisma API
- `staff-web/` — Vite + React staff-facing web app (Clinicians, Supervisors, Admins)
- `mobile/` — Expo + React Native patient-facing app
- `docs/` — product specs, screen specs, clinical protocols
